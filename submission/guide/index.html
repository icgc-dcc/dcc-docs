Guide

Overview

There are five major steps in the data submission process:

  1. Obtain login/password account from EGA's EBI repository/>
  2. Submit raw sequence data to the European Genome-phenome Archive
  3. Prepare the ICGC submission files according to DCC data format specifications
  4. Submit files to the DCC Secure FTP server
  5. Validate submission files using web-based submission system

For Release 15, all submitted data must be based on Human reference genome assembly GRCh37 and Ensembl gene set version 69.

When submitting experimental data to the ICGC DCC, please make sure you've already deposited your raw data to the appropriate public data repositories (eg: sequencing reads to EBI EGA). You will need to populate the data elements raw_data_repository and raw_data_accession with the correct repository and accession number respectively.

Please contact the DCC (_dcc-support@icgc.org) if you have any questions or comments about the data submission process._
